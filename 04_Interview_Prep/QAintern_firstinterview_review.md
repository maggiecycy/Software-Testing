### 3.1 OOP vs POP

POP (Procedural Programming):

* Function-centered
* Step-based logic
* Data and behavior separated
* Suitable for small systems

OOP (Object-Oriented Programming):

* Object-centered
* Encapsulation, inheritance, polymorphism
* Data and behavior bundled
* Better scalability and maintainability

Summary:

POP focuses on “how to do it”.
OOP focuses on “who does it”.

面向过程强调步骤。
面向对象强调角色与职责。

---

### 3.2 SQL CRUD

```sql
-- Create
INSERT INTO user(name, age) VALUES('Tom', 20);

-- Read
SELECT * FROM user WHERE age > 18;

-- Update
UPDATE user SET age = 21 WHERE name = 'Tom';

-- Delete
DELETE FROM user WHERE name = 'Tom';
```

Optimization knowledge:

* Add indexes
* Avoid SELECT *
* Use pagination
* Analyze execution plan using EXPLAIN

优化要点：

* 建索引
* 避免全字段查询
* 分页
* 使用 EXPLAIN 分析执行计划

### 3.4 Project Explanation Framework

### 项目讲解标准结构

Each project should answer:

1. What problem does it solve?
2. What tech stack was used?
3. What was my responsibility?
4. What challenges occurred?
5. How were they solved?
6. What was the final impact?
